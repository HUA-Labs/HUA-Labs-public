# @hua-labs/hua-ux

**Ship UX faster**: UI + motion + i18n, pre-wired.

A framework for React product teams that provides pre-wired UX defaults for spacing, components, motion, and internationalization.

## Quick Start

```bash
pnpm add @hua-labs/hua-ux
```

## Documentation

Full documentation coming soon. This is the Alpha version (0.x).

## Packages

- `@hua-labs/ui` - UI components
- `@hua-labs/motion-core` - Motion hooks
- `@hua-labs/i18n-core` - i18n core functionality
- `@hua-labs/i18n-core-zustand` - Zustand adapter for i18n
