# hua-ux Framework Documentation

hua-ux is a **meta-framework** for Next.js apps. It unifies UI (hua-ui), motion (motion-core), i18n (i18n-core), and state management into a single cohesive developer experience.

## Scope

This documentation covers **framework architecture and systems**. For individual component/hook APIs, refer to hua-docs site and `ai-docs/*.ai.yaml`.
