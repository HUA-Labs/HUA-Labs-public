## 3. Iconsax Icons (331개)

`components/icons/index.ts`에 정의된 Iconsax 아이콘.

### 사용법

```tsx
// IconProvider set="iconsax" 환경에서
<Icon name="Home2" />

// PROJECT_ICONS 매핑을 통해
<Icon name="home" /> // → Iconsax의 Home2로 변환
```

### 변형 (Variants)

모든 Iconsax 아이콘은 2가지 변형 지원:
- **Line** (기본) - 선으로 된 아이콘
- **Bold** - 채워진 아이콘
