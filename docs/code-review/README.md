# 코드 리뷰 가이드

이 폴더는 코드 리뷰 관련 문서들을 포함합니다.

## 문서 목록

- **[checklist.md](./checklist.md)** - 최근 코드 리뷰 체크리스트 (2025-12-04)
- **[fixes.md](./fixes.md)** - 코드 리뷰 수정 사항
- **[verification.md](./verification.md)** - 코드 리뷰 검증 결과
- **[general-checklist.md](./general-checklist.md)** - 일반적인 코드 리뷰 체크리스트 (Next.js 16, Tailwind CSS 4.0 마이그레이션 등)

## 코드 리뷰 프로세스

### 1. 리뷰 전 체크리스트 확인
코드 리뷰를 시작하기 전에 [checklist.md](./checklist.md)를 확인하여 주요 포인트를 파악하세요.

### 2. 리뷰 수행
- 코드 품질 확인
- 타입 안전성 확인
- 보안 취약점 확인
- 성능 이슈 확인
- 문서화 확인

### 3. 수정 사항 기록
발견된 문제점은 [fixes.md](./fixes.md)에 기록하고 수정을 진행하세요.

### 4. 검증
수정 완료 후 [verification.md](./verification.md)에 검증 결과를 기록하세요.

## 일반적인 코드 리뷰 체크리스트

### 코드 품질
- [ ] 코드가 프로젝트의 코딩 스타일을 따릅니다
- [ ] 타입 안전성이 보장됩니다
- [ ] 에러 처리가 적절합니다
- [ ] 주석이 필요한 부분에 주석이 있습니다

### 보안
- [ ] 사용자 입력 검증이 적절합니다
- [ ] SQL 인젝션 방지가 되어 있습니다
- [ ] XSS 방지가 되어 있습니다
- [ ] 인증/인가가 올바르게 구현되어 있습니다

### 성능
- [ ] 불필요한 리렌더링이 없습니다
- [ ] 데이터베이스 쿼리가 최적화되어 있습니다
- [ ] 메모리 누수가 없습니다

### 테스트
- [ ] 테스트가 작성되어 있습니다
- [ ] 모든 테스트가 통과합니다
- [ ] 엣지 케이스가 고려되어 있습니다
