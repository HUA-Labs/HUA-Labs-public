# 문서 아카이브

이 폴더는 더 이상 활발히 사용되지 않지만 참고용으로 보관하는 문서들을 포함합니다.

## 폴더 구조

### `devlogs/`
오래된 개발 로그 (2025-07, 2025-08)
- 초기 설정 및 개발 과정 기록
- 참고용으로 보관

### `completed-tasks/`
완료된 작업 관련 문서
- 마이그레이션 체크리스트
- 완료된 정리 작업 계획
- 과거 빌드 상태

### `deprecated/`
더 이상 사용되지 않는 문서
- 대체된 마이그레이션 가이드
- 사용하지 않는 프레임워크 가이드

### `temporary/`
임시 문서 및 예시 파일
- PR 템플릿 예시
- 임시 작업 계획

---

**참고**: 이 문서들은 참고용으로만 보관되며, 최신 정보는 `docs/` 폴더의 문서를 참조하세요.
