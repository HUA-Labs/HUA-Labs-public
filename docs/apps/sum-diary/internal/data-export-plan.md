# 데이터 내보내기 계획

> 최종 업데이트: 2025-11-07

## 개요

사용자가 자신의 일기 데이터를 다양한 형식으로 내보낼 수 있는 기능을 제공합니다.

## 지원 형식

- **HTML+CSS**: 웹 브라우저에서 볼 수 있는 형식 (ChatGPT 추천)
- **PDF**: 인쇄 및 보관용 PDF 형식
- **JSON**: 원본 데이터 형식
- **CSV**: 스프레드시트 분석용 형식

## HTML+CSS 형식 특징

- **반응형 디자인**: 모바일/데스크톱 최적화
- **인쇄 최적화**: 프린터 출력 지원
- **인터랙티브 요소**: JavaScript 기능 포함
- **스타일링**: 사용자 정의 CSS
- **인쇄 미디어 쿼리**: CSS print media queries

## 보안 고려사항

- **암호화 해제**: 내보내기 시 데이터 복호화
- **개인정보 보호**: 민감한 정보 필터링
- **접근 제어**: 본인 데이터만 내보내기 가능

## 구현 계획

1. **API 엔드포인트**: `/api/export` - 데이터 내보내기 API
2. **HTML 생성**: 서버 사이드 HTML 생성
3. **PDF 변환**: Puppeteer 또는 jsPDF 사용
4. **다운로드**: 파일 다운로드 기능

## 개발 단계

- **Phase 1**: HTML+CSS 형식 (우선순위 높음)
- **Phase 2**: PDF 형식
- **Phase 3**: JSON/CSV 형식
- **Phase 4**: 고급 기능 (필터링, 날짜 범위 등)
