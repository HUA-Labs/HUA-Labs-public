# 숨다이어리 개인정보 처리방침

## 📋 개요

숨다이어리는 사용자의 개인정보를 보호하고, 안전한 서비스를 제공하기 위해 다음과 같이 개인정보를 처리합니다.

## 🔒 수집하는 개인정보

### 1. 필수 수집 정보

#### 회원가입 시
- **이메일 주소**: 계정 식별 및 로그인
- **비밀번호**: 계정 보안 (해시화하여 저장)
- **닉네임**: 서비스 내 표시명

#### 서비스 이용 시
- **일기 내용**: 감정 분석 및 저장 (AES-256 암호화)
- **감정 분석 데이터**: AI 모델 개선용 (익명화)
- **사용 패턴**: 서비스 최적화용 (완전 익명화)

### 2. 자동 수집 정보

#### 기술적 정보
- **IP 주소**: 보안 및 어뷰즈 방지
- **User-Agent**: 디바이스 및 브라우저 식별
- **세션 ID**: 사용자 세션 관리
- **접속 시간**: 사용 패턴 분석

#### 서비스 이용 정보
- **일기 작성 시간**: 사용 패턴 분석
- **감정 분석 요청**: 서비스 사용량 측정
- **페이지 방문 기록**: 사용자 경험 개선

## 🎯 개인정보 활용 목적

### 1. 서비스 제공
- **일기 작성 및 저장**: 암호화된 일기 내용 보관
- **감정 분석**: AI 기반 감정 분석 서비스 제공
- **데이터 동기화**: 여러 기기 간 데이터 동기화

### 2. 서비스 개선
- **AI 모델 학습**: 익명화된 감정 데이터로 AI 성능 향상
- **사용자 경험 최적화**: 사용 패턴 분석을 통한 UI/UX 개선
- **서비스 안정성**: 시스템 모니터링 및 오류 개선

### 3. 보안 및 안전
- **어뷰즈 방지**: 부정 사용자 차단 및 보안 강화
- **계정 보안**: 로그인 시도 모니터링 및 보안 알림
- **법적 요구사항**: 관련 법령에 따른 정보 제공

## 🔐 개인정보 보호 조치

### 1. 암호화
- **일기 내용**: AES-256 암호화하여 저장
- **비밀번호**: bcrypt 해시화 (복호화 불가능)
- **개인 식별 정보**: 토큰화 또는 해시화

### 2. 익명화
- **분석용 데이터**: 개인 식별 정보 완전 제거
- **통계 데이터**: 집계된 형태로만 활용
- **AI 학습 데이터**: 개인 식별 불가능한 형태로 변환

### 3. 접근 제어
- **역할 기반 접근**: 직무에 필요한 최소한의 정보만 접근
- **감사 로그**: 모든 데이터 접근 기록 보관
- **정기 감사**: 분기별 보안 감사 실시
