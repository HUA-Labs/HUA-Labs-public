# HUA Animation Site 🎬

**HUA Labs의 애니메이션 라이브러리를 체험하고 문서를 확인할 수 있는 공식 사이트입니다.**

## ✨ 주요 기능

### 🎯 **애니메이션 갤러리**
- **모든 애니메이션 효과**의 라이브 데모
- **실시간 성능 모니터링** (FPS, 메모리 사용량)
- **인터랙티브 체험** - 마우스/터치로 애니메이션 제어

### 📚 **문서화**
- **초보자 친화적** 가이드
- **실용적인 예제** 코드
- **성능 최적화** 팁

### 🎮 **플레이그라운드**
- **실시간 코드 편집**
- **애니메이션 조합** 실험
- **코드 공유** 기능

## 🚀 시작하기

### 개발 서버 실행
```bash
pnpm dev
```

### 빌드
```bash
pnpm build
```

### 프로덕션 실행
```bash
pnpm start
```

## 🛠 기술 스택

- **Next.js 15** - React 프레임워크
- **TypeScript** - 타입 안전성
- **Tailwind CSS** - 스타일링
- **HUA Animation** - 애니메이션 라이브러리
- **HUA UI** - 디자인 시스템
- **HUA i18n** - 다국어 지원

## 📁 프로젝트 구조

```
src/
├── app/
│   ├── components/     # 페이지별 컴포넌트
│   ├── docs/          # 문서 페이지
│   └── playground/    # 플레이그라운드
├── components/        # 공통 컴포넌트
└── translations/      # 다국어 파일
```

## 🎬 애니메이션 시스템

### 기본 애니메이션
- **FadeIn/FadeOut** - 페이드 효과
- **SlideUp/SlideDown** - 슬라이드 효과
- **ScaleIn/ScaleOut** - 크기 변화
- **BounceIn/BounceOut** - 바운스 효과

### 고급 애니메이션
- **Spring** - 스프링 물리 효과
- **Gradient** - 그라데이션 애니메이션
- **Repeat** - 반복 애니메이션
- **Interactive** - 인터랙티브 애니메이션

### 성능 최적화
- **60fps 보장** - 부드러운 애니메이션
- **메모리 효율성** - 자동 정리
- **성능 모니터링** - 실시간 측정

## 🌍 다국어 지원
