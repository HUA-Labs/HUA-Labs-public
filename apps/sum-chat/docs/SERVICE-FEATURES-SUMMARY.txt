[HUA(숨챗) 서비스 대표 기능 요약]

1. 감응 프리셋 기반 챗봇
   - tone(톤), mode(모드), tier(티어) 등 감정/상태 프리셋 적용
   - 대화 맥락에 따라 자동/수동 감응값 변화

2. 세션/사이드바/상태 관리
   - 세션 생성/삭제/선택, 세션별 대화 기록
   - 사이드바에서 세션 목록 및 상태 확인

3. 자동 세션 제목 생성
   - 첫 메시지 입력 시 GPT 기반 세션 제목 자동 생성

4. 감응값(GPT) 자동 분석 및 표시
   - 대화마다 tone/mode/tier 등 감응값 자동 추출 및 상태 반영

5. 윤리/보안 정책 내장
   - 위반 감지 시 tier 0.0(차단) 자동 전환, 알림/로그 기록

6. 숨일기 연동
   - 대화 내용을 "숨일기"로 전송, 감정/상태 분석 및 저장

7. 감응 데이터 시각화/분석(준비 중)
   - 유저별 tone/mode/tier 비율 등 데이터 기반 피드백/추천

8. 확장성
   - 감응 기반 음악 추천(숨송), 빅데이터 분석 등 다양한 서비스 연동 준비

[기술 스택]
- Next.js(React), TypeScript, YAML 프리셋, OpenAI GPT API, LocalStorage, API Routes 등
