## Logo 컴포넌트 수정

### 문제
- `variant="full"` (엠블럼 + 워드마크) 사용 시 로고가 겹쳐서 보임
- 워드마크 SVG의 viewBox 높이가 37인데 엠블럼은 40

### 수정
- viewBox `40 0 152 37` → `40 0 152 40` (높이 통일)
- width 계산: `calc(${sizeInRem} * 3.8)` (152/40 비율)

## 향후 계획: @hua-labs/ui-patterns

### 개념
- **Atomic → Compound**: 현재 `@hua-labs/ui`는 atomic 컴포넌트 (Button, Card, Input 등)
- **ui-patterns**: 폼 중심의 compound 컴포넌트 패키지

### 예시 패턴
- `LoginForm` - 이메일/비밀번호 + 소셜 로그인
- `SignupForm` - 가입 폼 + 유효성 검사
- `ContactForm` - 문의 폼
- `SearchWithFilters` - 검색 + 필터 패널
- `DataTable` - 정렬/필터/페이지네이션 테이블
- `SettingsPanel` - 설정 그룹화 패널

### 장점
- DX 향상: 반복 패턴을 즉시 사용
- 일관성: 동일한 UX 패턴 보장
- 커스터마이징: props/slots로 확장 가능
